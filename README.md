# 专注计时器应用

一个具有科技感和液态玻璃效果的专注计时应用，支持计时、活动记录和浮窗显示功能。

## 功能特点

- ⏰ 精准计时器，支持分秒显示
- 📝 活动输入，实时放大显示
- 📋 活动记录保存
- 🪟 浮窗显示，可在其他应用上查看计时状态
- 🎨 液态玻璃质感界面，科技感强
- 📱 响应式设计，适配手机屏幕

## 本地运行

1. 进入项目目录
2. 启动本地服务器
   ```bash
   # 使用Python 3
   python -m http.server 8000
   
   # 或使用Python 2
   python -m SimpleHTTPServer 8000
   ```
3. 在浏览器中访问 http://localhost:8000

## 部署到免费托管平台

### 国外平台

#### 方法1: GitHub Pages

1. 在GitHub上创建一个新的仓库
2. 将项目文件上传到仓库
3. 在仓库设置中启用GitHub Pages，选择main分支
4. 等待部署完成，获得访问链接

#### 方法2: Vercel

1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击"New Project"
4. 选择"Import Git Repository"或直接拖放项目文件夹
5. 按照提示完成部署

#### 方法3: Netlify

1. 访问 https://netlify.com
2. 使用GitHub账号登录
3. 点击"Add new site"
4. 选择"Deploy manually"
5. 拖放项目文件夹到指定区域
6. 等待部署完成

### 国内平台（推荐）

## 技术栈

- HTML5
- CSS3 (使用backdrop-filter实现液态玻璃效果)
- JavaScript (原生)
- 本地存储 (localStorage)

## 界面特色

- 液态玻璃质感：使用backdrop-filter和渐变实现
- 实时活动显示：输入框在有文字时自动放大变色
- 未来感数字：使用Orbitron字体和文字阴影
- 流畅动画：页面切换和按钮交互效果
- 响应式设计：适配不同屏幕尺寸

## 使用说明

1. 在顶部输入框中填写你正在做的事情
2. 点击中央的绿色按钮开始计时（再次点击暂停）
3. 点击右侧的记录按钮查看历史记录
4. 浮窗功能可在其他应用中查看计时状态

希望这个应用能帮助你提高专注力和工作效率！
